  • UG_COSEAT_3 AI Report Summarizer - SummieAI (9)

    Students: Christine Khai Ning Bong, Daneel Hewitt, Ameer Musharraf Noor Musaddik, Kanzen Ong, Movindu Wijenayake, Josh Bakanursky

    Supervisor: Bao Vo

    Client: Violetta Misiorek , Suncorp Group

    This project aims to develop a LLM powered system that automates the summarisation of reports provided to the system. The system analyses both textual content and graphical data, extracting and combining key insights such as trends, performance drivers, and key findings into clear, meaningful summaries. Users can upload or select reports through an intuitive interface and receive structured summaries that highlight the most relevant information. The system also allows dynamic customisation of the summarisation process, enabling users to adjust factors such as summary focus and intended audience to better suit their needs. Additionally, the solution prioritises and ranks extracted insights based on their relevance and impact, helping users quickly identify the most important points. Overall, the system is designed to provide faster, more consistent, and user-tailored summaries, significantly reducing the time and effort required for manual report analysis.

    Students: Connor Lack, Jen Mao, Jason Vo, Kavindu Bhanuka Weragoda

    Supervisor: Vasudha Malhotra

    Client: Tanjila Kanij, Department of Computing Technologies

    UG_COSEAT_18 FloodAI: An Agentic AI System for Real-Time Flood Hazard Triage in Urban Catchments (7)

    Students: Muhammad Alwaz, Austin Cook, Arsalan Hasan, Yuk Hei Ho, Binara Kavihara Rambukkanage, Xiaosu Wang

    Supervisor: Waqar Khan

    Client: Abdul Aziz, Frankston City Council

    FloodAI is an AI-assisted flood risk triage system developed for Frankston City Council as part of a Swinburne University capstone project. The system allows residents to enter their property address and receive a real-time flood risk assessment based on live Bureau of Meteorology rainfall data, twelve months of historical ground moisture records, and pre-simulated TUFLOW flood models covering the Frankston South catchment. An AI agent classifies flood hazard from H1 to H6 per Melbourne Water guidelines and generates plain-English guidance aligned with VicEmergency warning levels. The system is accessible through both a Streamlit dashboard and a React web application with an interactive flood simulator.

  • UG_COSEAT_19 AI Agent for Flood Assessment Report Automation (6)

    Students: Anshul Vats, Parthasarathi Jayaprakash, Justin Nguyen, Dhruv Pravinbhai Patel, Leonardo Pietralla, Vishal Vinod

    Supervisor: Rao Liaqat

    Client: Abdul Aziz, Frankston City Council

    This project explores the development of an AI-assisted system to automate flood assessment report writing. It involves analyzing standard council report structures, interpreting hydrologic and hydraulic model outputs (such as TUFLOW data), and applying natural language processing with large language models to generate coherent report narratives. The system will be refined to match professional tone and include visual elements like maps and charts. The final prototype will integrate with tools such as QGIS or Office 365, enabling engineers to produce consistent, accurate, and time-efficient flood reports while improving accessibility of flood risk information for communities.

  • UG_COSEAT_27 Automating the Alignment of a Laser Beam through an Optical Fibre using Reinforcement Learning (5)

    Students: Dev Bohra, Morgan Caddy, Aaron Dimovski, Thanh Nghia Do, Anh Luu, Robin Sidhu

    Supervisor: Mukesh Malani

    Client: Sascha Hoinka, School of Science, Computing and Emerging Technologies, Swinburne

    This project aims to automate the alignment of laser beams through optical fibres for quantum gas experiments using reinforcement learning. The system combines stepper motors, photodiode feedback and a microcomputer to continuously optimise laser coupling efficiency with minimal human intervention. By replacing manual mirror adjustments with an intelligent control system, the project seeks to improve reliability, reduce alignment time, and maintain stable laser output. The prototype will include custom 3D-printed mounts, motor control electronics, remote monitoring, and a live UI dashboard for displaying alignment data and system status, creating a scalable and cost-effective solution for advanced optical research laboratories.

  • UG_COSEAT_32 A Model-Based Design Interface for Quantum ML Pipeline Generation (8)

    Students: Sampathawaduge De Silva, Abid Us Sobhan, Shahbaz Taseen, Maleesha Tissera

    Supervisor: Waqar Khan

    Client: Prem Prakash Jayaramen, Swinburne University of Technology

    This project develops a visual Quantum Machine Learning (QML) pipeline builder using Model-Based Design principles. The application allows users to create QML workflows through connected visual blocks representing datasets, encoders, quantum circuits, optimisers, and outputs. The system automatically converts the visual pipeline into a structured JSON specification and generates executable Qiskit and PennyLane code templates. The application also supports execution using the Qiskit Aer quantum simulator and demonstrates use cases in Finance, Human Resources, and Supply Chain domains. The project aims to simplify QML experimentation, improve accessibility to quantum computing concepts, and provide an educational and research-oriented development platform.

  • UG_COSEAT_34 Simulating Radiation Effects on In-Orbit Convolutional Neural Networks (8)

    Students: Mika Ko, The Son Ngo, Viet Hoang Pham

    Supervisor: Humphrey Obie

    Client: Sam Hall, Swinburne University of Technology

    Advances in satellite imaging have increased data volumes, making onboard ML processing, particularly CNNs, attractive for reducing space-to-ground transmission. However, the radiation environment in orbit poses serious risks to electronics, especially Commercial-off-the-Shelf (COTS) hardware. Charged particles cause Single Event Upsets (SEUs), bit-flips that can unpredictably degrade CNN performance. This project designs a simulation framework to model radiation-induced SEU impacts on CNN inference accuracy across different orbital altitudes, identify the most vulnerable layers and components, and recommend targeted radiation-hardening strategies that maximise reliability while minimising cost.
